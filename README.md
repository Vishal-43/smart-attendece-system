# Smart Attendance System

Multi-platform attendance management system using geo-fencing, QR codes, and access point matching.

## Tech Stack
- **Backend:** Python (FastAPI), Java (Spring Boot), Haskell (Servant)
- **Frontend:** React (TypeScript), Flutter (Dart)
- **Database:** Supabase (PostgreSQL)
- **Cache:** Redis
- **DevOps:** Docker, GitHub Actions, Kubernetes
